# ChatGPT 对话导出与复用教程

## 一、安装篡改猴插件

1. 打开浏览器的**扩展商店**（Chrome 叫"Chrome 应用商店"，Edge 叫"Microsoft Edge 扩展"），搜索 **篡改猴**，点击安装
2. 进入浏览器的扩展管理页面，打开**开发者模式**
3. 找到篡改猴插件，点击**详情**，把"**允许运行用户脚本**"的开关打开

## 二、安装导出脚本

打开脚本安装链接：

- **稳定版**：`https://raw.githubusercontent.com/dofgm/ChatGPT_Exporter/main/chatgpt-exporter.user.js`

篡改猴会自动弹出安装页面，点击**安装**就好了。

## 三、导出对话

1. 打开 ChatGPT 网站，已登录的**刷新一下页面**，没登录的先登录
2. 页面右下角会出现一个绿色的 **Export** 按钮
3. 点击按钮，选择格式和范围，等待导出完成，ZIP 文件会自动下载

## 四、导出的三种格式分别干嘛用

| 格式 | 用途 |
|------|------|
| **HTML** | 本地查看用，双击浏览器打开就能看，带样式带代码高亮 |
| **Markdown** | 当附件上传到新的 ChatGPT 对话中，让 AI 读取后继续聊 |
| **JSON** | 给程序员用的，写代码做数据分析、批量处理 |

## 五、在新对话中复用

1. 新建一个 ChatGPT 对话
2. 点 📎 附件，上传之前导出的 **.md 文件**
3. 告诉 AI 你要接着聊什么，比如："这是我之前的对话记录，请继续帮我完善里面提到的方案"
4. AI 会读取附件内容，基于之前的上下文继续工作
